Cascading Style Sheets ;linkene
 

Her finner du koden jeg har brukt for å lage en dynamisk effekt på linkene. Når du peker og klikker på en link, får den en dekorativ linje under teksten.

I tillegg til at denne linjen er "pen", gir den en visuell tilbakemelding om at dette er en link som kan klikkes på. Det gjør at siden blir lettere å bruke, fordi man ser bedre hva som er linker og hva som er vanlig tekst.

Koden er veldig lett. Den eneste forskjellen fra annen bruk av Cascading Style Sheets jeg har vist, er at man bruker et kolon i klassenavnet for å definere de ulike tilstandene link-elementet er i.

  • a:link angir en vanlig link
  • a:hover angir at linken pekes på
  • a:active angir at linken klikkes på
  • a:visited angir at man tidligere har besøkt siden det linkes til

    a:link
    {
       text-decoration: none;
       color: #FFAD00;
       font-size: 8pt;
       font-weight: 800;
       font-family: 'Verdana, Arial, Helvetica, Sans-Serif'
    }

    a:hover
    {
       text-decoration: underline;
       color: #FFAD00;
       font-size: 8pt;
       font-weight: 800;
       font-family: 'Verdana, Arial, Helvetica, Sans-Serif'
    }

    a:active
    {
       text-decoration: none;
       color: #FFAD00;
       font-size: 8pt;
       font-weight: 800;
       font-family: 'Verdana, Arial, Helvetica, Sans-Serif'
    }

    a:visited
    {
       text-decoration: none;
       color: #FFAD00;
       font-size: 8pt;
       font-weight: 800;
       font-family: 'Verdana, Arial, Helvetica, Sans-Serif'
    }
